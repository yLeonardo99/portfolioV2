• Flex Jobs

Plataforma de Freelancers

Esse foi meu primeiro projeto, foi a criação de um website com o objetivo de ser uma plataforma para freelancers. O projeto consistia em desenvolver um protótipo de um site de vagas de empregos para freelancers, e ao longo do desenvolvimento, evoluímos para incluir funcionalidades que permitiram o envio de dados de cadastro para o banco de dados. Para o desenvolvimento, utilizamos o XAMPP como servidor local, simulando o ambiente de servidor no computador.

• Pet Friends

ONG

Este projeto, criado para a ONG PetFriends, tem como propósito desenvolver um site informativo e interativo voltado à adoção de cães. O principal objetivo é sensibilizar a comunidade sobre a relevância da adoção responsável, além de oferecer um espaço para que as pessoas conheçam os animais disponíveis para adoção e descubram formas de apoiar a causa.
